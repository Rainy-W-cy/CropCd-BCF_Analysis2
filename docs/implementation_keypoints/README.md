# 实现重难点文档索引

本目录用于沉淀 `soil_task_project` 的关键实现逻辑，内容严格基于当前代码（`src/` + `notebooks/`）整理。

## 文档清单

1. `01_流程总览与实验方法.md`
   - 端到端实验链路（01~08）
   - 两条评估路线：空间交叉验证 vs 7:3 Holdout
   - 输出物与目录关系

2. `02_src重点解析.md`
   - `src` 各核心模块职责
   - 关键函数输入/输出与实现要点
   - 易踩坑行为（如 holdout 的训练集混入评估集）

3. `03_notebook重点解析_参数与SHAP最优模型.md`
   - 每个 notebook 的真实调用逻辑
   - 参数调节入口（`config.yaml`）
   - SHAP 选“全局最优模型”的具体规则与过滤逻辑

## 使用建议

- 新成员先读 `01`，再读 `03`（和实验执行最相关），最后按需查 `02`。
- 每次改动 `src` 关键函数或 notebook 主流程后，同步更新本目录。
