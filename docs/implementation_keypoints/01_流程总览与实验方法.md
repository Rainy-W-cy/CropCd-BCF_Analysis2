# 流程总览与实验方法（基于当前代码）

## 1. 端到端流程（notebooks）

1. `01_data_prep.ipynb`
   - 调用 `src.data_io.read_excel_as_long` 读取 Excel（支持多 sheet 合并）。
   - 调用 `add_ph_bins` 依据 `config.yaml -> ph_bins.edges/labels` 生成 `ph_bin`。
   - 调用 `src.audit.enforce_required_columns` 强校验目标列、pH、X、Y。
   - 输出：`outputs/clean_data.parquet`、`outputs/data_audit.xlsx`。

2. `02_qc_optional.ipynb`
   - 调用 `src.qc_optional.run_optional_qc`。
   - 输出缺失率、数值统计、BCF 分布图、分组统计与 QC 备注。

3. `03_eda_feature.ipynb`
   - 生成分组描述统计与数值相关热图（基础版）。
   - 输出：`outputs/eda_tables.xlsx`、`outputs/figures/corr_heatmap_numeric.png`。

4. `03b_corr_advanced.ipynb`
   - 调用 `src.corr_advanced.build_group_correlations` 按分组输出论文风格相关图。
   - 调用 `export_corr_pdf` 汇总 PDF。

5. `04_modeling_spatialcv.ipynb`
   - 调用 `src.spatial_cv.make_spatial_groups` + `GroupKFold` 做空间交叉验证。
   - 调用 `src.models.fit_oof_with_spatialcv` 训练并输出 OOF 指标。
   - 输出：`outputs/metrics_spatialcv.xlsx` 与 `outputs/figures/obs_pred_*.png,resid_*.png`。

6. `04b_holdout_70_30.ipynb`
   - 调用 `src.holdout.run_holdout` 做 7:3 切分评估（按组或全局由 config 控制）。
   - 输出：`outputs/holdout_70_30/metrics_holdout.xlsx` 等。

7. `05_shap_new.ipynb`
   - 先读取 `metrics_holdout.xlsx`，按加权指标选“一个全局模型”。
   - 再用该单一模型对所有组做 SHAP（非空间 CV）。
   - 输出 SHAP 图、特征贡献表、模型对比柱状图。

8. `06_thresholds_gb.ipynb`
   - 当前代码优先用 `xgb`，否则 `rf` 进行阈值反推（不是沿用 SHAP 的全局最优模型）。
   - 调用 `src.thresholds.infer_soil_cd_thresholds` 做二分求解阈值。

9. `07_reports_pdf.ipynb`
   - 调用 `src.reporting.export_oneclick_pdf` 汇总 outputs 为 PDF。

10. `08_maps.ipynb`
   - 直接 `import src.maps as maps_mod`，调用 `maps_mod.make_maps`。
   - 会强制检查 `geopandas`、`pyshp`、`boundary_path` 是否可用。

## 2. 两条实验评估路线

1. 空间交叉验证路线（`04`）
   - 方法：`GroupKFold`，分组来自 `X/Y` 的网格或 kmeans 空间分箱。
   - 优点：更贴近空间外推评估。

2. Holdout 路线（`04b`）
   - 方法：7:3（`test_size=0.30`）切分。
   - 注意：默认 `mix_train_into_test=true`，会把训练样本混入评估集，指标可能偏乐观。

## 3. 输出目录关键关系

- 数据与审计：`outputs/clean_data.parquet`, `outputs/data_audit.xlsx`
- 空间CV：`outputs/metrics_spatialcv.xlsx`, `outputs/figures/*.png`
- Holdout：`outputs/holdout_70_30/*`
- SHAP：`outputs/shap/*`
- 地图：`outputs/maps/*`
- 汇总报告：`outputs/reports/oneclick_report.pdf`
